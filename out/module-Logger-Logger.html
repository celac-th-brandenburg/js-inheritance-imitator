Class: Logger

Logger~Logger()

Logs messages with the browser Console API. This class should not be instantiated and is supposed to be used as a constructor function to access static methods.

Constructor

new Logger()

The Logger class only provides static methods and should not be instantiated.

Calling the constructor function of Logger throws an error.
