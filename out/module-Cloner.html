Module: Cloner

Performs extension & multiple inheritance operations between an inheriting instance, object or prototype (host object) and one or more inherited instances, objects, prototypes or constructor functions (extension object). This class should not be instantiated and is supposed to be used as a constructor function to access static methods.

(static) extend(hostObject, extensionObject, config, …extensionConstructorArgs) → {Boolean}

creates an inheritance realtionship between two objects based on the provided configuration
Parameters:
Name Type Attributes Description
hostObject Object the inheriting (scope) host object; can be an instance or a prototype
extensionObject Object the inherited extension object; can be an instance, a prototype or a constructor function object (class)
config Object the configuration object; might contain some or none of the following fields: register, mirrorGettersAndSetters, overrideGettersAndSetters, mirrorFunctions, mirrorOthers, overrideFunctions, overrideOthers
Properties
Name Type Description
register Boolean whether or not the extension object should be registered in the __extends Set of the host object
mirrorGettersAndSetters Boolean whether or not accessor descriptors (getter and setter methods) should be mirorred/replicated/copied/imitated/mimiced/simulated
overrideGettersAndSetters Boolean whether or not accessor descriptors (getter and setter methods) should be overriden
mirrorFunctions Boolean whether or not data descriptors with a function as a value (methods) should be mirorred/replicated/copied/imitated/mimiced/simulated
overrideFunctions Boolean whether or not data descriptors with a function as a value (methods) should be overriden
mirrorOthers Boolean whether or not data descriptors that do not have a function as a value (data descriptor attributes that are not methods) should be mirorred/replicated/copied/imitated/mimiced/simulated
overrideOthers Boolean whether or not data descriptors that do not have a function as a value (data descriptor attributes that are not methods) should be overriden
extensionConstructorArgs <repeatable>
 further arguments/parameters to be passed to the constructor of the inherited extension object
Since:

the success of the operation
